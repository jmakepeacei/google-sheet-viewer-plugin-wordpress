== Description ==

Este plugin muestra los datos de una hoja de cálculo de Google Sheets y la visualiza a través de un ShortCode en Wordpress.

== Installation ==

1. Upload `portfolio-plugin` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[google_sheet_data]` shortcode to display the plugin
